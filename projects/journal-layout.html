Project Overview

Objective

Design and produce a professional, publication-ready magazine feature for Design Through The Decades Journal that integrates editorial writing, typographic hierarchy, and grid-based layout systems. The project focused on translating long-form content into a cohesive visual narrative using industry-standard editorial design principles.

My Role

Lead editorial designer and content developer responsible for layout design, typography, visual system development, and content integration.

Tools

Adobe InDesign, Adobe Illustrator/Photoshop, grid and type system frameworks, APA citation formatting.

Deliverables

16-page magazine including front/back cover, multi-spread feature article, visual appendix, typographic system, images, and fully formatted references.

Process

Approach

  1. Concept Development & Refinement: Revised and structured a long-form article, incorporating peer feedback and organizing content for editorial flow across multiple spreads.
  2. Typography & System Design: Selected and applied a consistent type scale to establish hierarchy for headlines, subheads, body text, captions, and supporting elements.
  3. Grid & Layout Exploration: Designed multiple spread layouts using varied grid systems to balance text and imagery while maintaining visual consistency.
  4. Layout Execution: Built the full magazine, aligning all elements to grids, refining spacing, and ensuring readability through proper text flow and alignment.
  5. Polish & Quality Control: Eliminated widows/orphans, refined visual hierarchy, ensured consistent styling, and finalized a cohesive, publication-ready design.
